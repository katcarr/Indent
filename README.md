Indent
==============

A simple ruby on rails blog app. Allows a user to add blog posts and for others to comment on those posts.

Installation and Usage
------------
Run Postrges Sever

Clone or download this repository and cd into indent directory

```
$ rake db:create
$ rake db:migrate
$ rails server
```

In your browser visit localhost:3000

or check use the app at

http://indent.herokuapp.com
